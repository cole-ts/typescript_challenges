# typescript_challenges

A repository for Coles typescript solutions to the weekly challenges from the Terasense Team. 

## Running the tests

```bash
npm run test
```

## Running the tests in watch mode

```bash
npm run test:watch
```

### Challenges:

1. [Palindrome](src/palindrome/README.md)
    - [src/palindrome/palindrome.ts](src/palindrome/palindrome.ts) - contains the solution to the palindrome problem
    - [src/palindrome/palindrome.test.ts](src/palindrome/palindrome.test.ts) - contains the tests for the palindrome problem
